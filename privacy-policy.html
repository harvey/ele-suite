Privacy Policy — ELE Suite Browser Extension

Last updated: December 28, 2025

Summary

This extension enhances Panopto/course pages with playback speed controls, lecture completion checkboxes, and a notes link feature. It collects minimal data required to provide those features and gives you control over what is stored or synced.

Data We Collect

  • Logged-in email (optional): The extension attempts to read the currently logged-in user's email from the locally accessible /user/profile.php page in order to enable optional cloud syncing of your data. If it cannot find an email, cloud sync is not attempted.
  • Lecture completion state: When you check or uncheck a lecture, the extension stores a simple completed flag per lecture URL.
  • Notes links: If you attach a notes link to a lecture (a URL you provide), that link is stored and may be synced if you enable cloud sync.
  • Local UI settings: Playback speed preferences are stored in the browser's localStorage (key: panopto_custom_speed) and some UI state is stored in the extension's local storage (e.g., completion data under a key like panopto-completed-<courseId>).

Where Data Is Stored

  • Local browser storage: Completion state and notes are stored locally using the extension storage (Chrome/Chromium extension storage) and localStorage. This data remains on your device until you clear it or remove the extension.
  • Optional cloud sync: If an email is available, the extension offers an optional cloud sync feature that posts/gets your data to an external sync endpoint operated by the extension author. The service stores only the completion flags and notes links associated with the provided email. Cloud sync is optional and can be disabled by not providing an email-accessible profile or by clearing sync data.

How We Use Your Data

Collected information is used only to:

  • Display and persist which lectures you have marked as completed.
  • Store and open notes links you attach to a lecture.
  • Apply playback speed settings (local only) and, if enabled, sync your completion/notes between devices via the optional cloud endpoint.

Third-Party Services

The optional cloud sync uses a remote endpoint to save and retrieve your completion/notes data. If you enable that feature, your data will be transmitted to and stored on that service. The extension does not automatically share your data with any other third parties.

Security

We take reasonable measures to protect data transmitted to the optional cloud sync service, but no transmission over the internet is 100% secure. Notes links you save may reference resources that require login — if you want them to remain private, prefer storing links that require account-based access or avoid using cloud sync.

Your Choices

  • Disable cloud sync: Do not allow the extension to access a profile email (or remove any stored email) to avoid cloud sync attempts.
  • Remove local data: You can clear the extension's stored data via your browser's extension storage settings or by clearing site/local storage used by the extension. Clearing localStorage will remove playback speed preferences (key: panopto_custom_speed).
  • Delete notes & completions: You can uncheck items to remove completion flags locally. To remove notes links, edit or delete them via the extension UI. If you have used cloud sync, you can delete synced data by clearing the stored data in the extension and (if supported) using the cloud sync UI or contacting the operator (see Contact below).
  • Uninstall: Removing the extension deletes its local storage; check your browser's extension removal flow for details.

Retention

Local data is retained until you clear it or uninstall the extension. Data stored on the optional cloud sync service is retained until you or the operator delete it.

Contact

If you have questions about this privacy policy, data stored by the extension, or would like your cloud-synced data removed, contact the extension operator at: ele-suite@local (replace with the operator's actual contact email if desired).

Changes to This Policy

We may update this policy from time to time. When we do, we will update the "Last updated" date at the top of this page.

Note: This policy describes how the ELE Suite extension behaves based on the extension's source code. It is provided for informational purposes and does not replace any platform or store policies.
